Project: Vaalikartta

2019-12-27

Vaalikartta is a web app to explore data about Finnish parliamentary elections from 1983 to 2019. It also includes some information about each municipality and cabinets.

It is a product of the course Open Data Student Projects (TIEA207). We developed the app in a group of four students. I took care of extracting most of the data and developing the back-end API. We didn’t have any designated project manager, but our project board was managed mostly by me. Also, I was responsible for automating of the deploying process, so I guess I was the teams DevOps guy too.

To build the project we used the following technologies:

  • React, D3, Bootstrap
  • Node.js
  • MongoDB
  • Google App Engine
  • Github

The source code is available on Github.
