Project: Homepage

2019-09-04

I started working on this site on 2019-09-04. The site is built with Hugo using its various functionalities (eg. templates). The theme is my homebrew. I call it minimus for being minimal (although barebones would describe it equally well).

As a side note, Hugo is based on the Go programming language, which I haven’t particularly used at all, but it seemed the best alternative of the static site generators I know of (Jekyll, Hakyll, Gatsby.js, Pelican). It took some work to wrap my head around how to make it work for me. Surely it would have been easier to use a premade theme, but where’s the fun in that? üòÅ
